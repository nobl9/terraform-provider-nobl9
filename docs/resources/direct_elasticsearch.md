# nobl9_direct_elasticsearch (Resource)

Elasticsearch is a distributed search and analytics engine. Nobl9 connects to Elasticsearch for SLI measurement collection and comparison with SLO targets.

For more information, refer to [Elasticsearch Direct | Nobl9 Documentation](https://docs.nobl9.com/Sources/elasticsearch#elasticsearch-direct).

## Example Usage

```terraform
resource "nobl9_direct_elasticsearch" "example" {
  name            = "elasticsearch-direct"
  project         = "default"
  url             = "https://example.aws.found.io"
  api_key         = "encoded-api-key"
  release_channel = "stable"

  historical_data_retrieval {
    default_duration {
      unit  = "Day"
      value = 1
    }
    max_duration {
      unit  = "Day"
      value = 30
    }
  }

  query_delay {
    unit  = "Minute"
    value = 1
  }
}
```
